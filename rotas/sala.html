Destaque

Breve para começar

Esta é a web biblioteca LECP (Livros Educacionais e Conhecimento Priorizado), um lugar para você ler livros difíceis de encontrar. Toda a bilioteca está contida em drives, o que torna este site apenas um guia, como um sumário. Aproveite!

  • Propósito

    Este site foi criado para o bem e com objetvo de ajudar os estudantes.

  • Estética

    O design foi pensado em aplicações de filmes, mas com livros sendo o conteúdo.

  • Dúvidas

    Em caso de dúvidas, mande uma mensagem! A sua ideia também possui valor.

Sobre os livros

Muitos alunos não tem condições de comprar livros, ou não moram próximo de uma biblioteca, então seria justo entregar de forma virtual o material que eles procuram. O conhecimento é para todos.
